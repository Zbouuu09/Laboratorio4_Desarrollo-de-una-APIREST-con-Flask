# TecnoMarket - API REST de Catálogo de Productos (Laboratorio 4)

Este proyecto implementa una API REST básica para la gestión en memoria de un catálogo de productos de la empresa **TecnoMarket** utilizando **Flask** en Python.

## Estructura del Proyecto
* `app.py`: Servidor backend principal con los endpoints REST.
* `cliente.py`: Script para pruebas automatizadas de peticiones.
* `requirements.txt`: Listado de librerías y dependencias.
* `.gitignore`: Configuración para excluir carpetas virtuales de Git.
